Adcovery's plugin keeps your Adcovery Third Party Init code up to date!

== Description ==

Adcovery adds automatic synchronization to all your Adcovery widgets.

== Installation ==

1. Install the Adcovery plugin by uploading the files to your server or by downloading the plugin on https://wordpress.org/plugins/.
2. Activate the newly installed Adcovery plugin.
3. Go to the settings page and fill in your designated Adcovery Website ID and API Key.
4. Make sure the "Enabled" checkbox is checked.
5. Click "Save & Refresh".
6. Installation is complete.

== Frequently Asked Questions ==

= Where do I find my Adcovery Website ID and API key? =

Get in touch with the Adcovery team to request your credentials: contact@adcovery.com

= I have my Website ID and API key, what should I do with them? =

Make sure the Adcovery plugin is installed and activated. Go to Settings > Adcovery and insert them in the form displayed on the page, tick the 'enabled' checkbox, and save!
